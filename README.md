# Running it locally

1. Clone the repo

   ```sh
   git clone https://github.com/retconned/portfolio
   ```

2. Change directory to the project folder

   ```sh
   cd portfolio
   ```

3. Run the app locally

   ```sh
   yarn dev
   ```

# Stack

- Next.js
- Typescript
- TailwindCSS
- Vercel
